## To-Do App

A modern to-do list application built with React and Vite to help you create, organize, and track your daily tasks efficiently.

- **Live Demo**: [To-Do App on Netlify](https://cute-pixie-f4fc9a.netlify.app/)

---

### Features

- **Add tasks**: Quickly create new to-do items.
- **Mark as complete**: Track progress by completing tasks.
- **Clean UI**: Simple and responsive interface focused on usability.

---

### Tech Stack

- **Framework**: React
- **Build Tool**: Vite
- **Language**: JavaScript
- **Deployment**: Netlify (`https://cute-pixie-f4fc9a.netlify.app/`)

---

### Getting Started

#### Prerequisites

- **Node.js** (LTS version recommended)
- **npm** or **yarn**

#### Installation

```bash
git clone <your-repo-url>
cd to-do-app
npm install
```

#### Development

```bash
npm run dev
```

Then open the URL shown in your terminal (usually `http://localhost:5173`).

#### Production Build

```bash
npm run build
```
